to_do_list

  1. 切换列表项开/关。
    • 简单的在 [ ] 和 [X] 之间切换。
    • 所有的列表子项将被适当的切换为开/关。
    • 仅当当前行是列表项时，切换子项。
    • 父列表项将受子列表项的切换影响。
  2. 同样支持编号列表项的切换。
  3. 添加高亮到列表项的复选框。
  4. 使用 o, O 和 <CR> 创建新行时，新列表项会带 [ ]。(译注：<CR>没有带 [ ])
